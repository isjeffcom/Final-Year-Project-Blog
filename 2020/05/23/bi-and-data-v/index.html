Business Intelligence

Data-driven business intelligence(BI) is a phenomenon topic along with data science and data visualization technology. When business dealing with big data, especially when the data is too big for any human can interpret without using a certain visualization method.

Despite there is a business in the word BI, this technology is not only work in the industrial. Let made an example of the COVID-19 epidemic, in the year of the Internet, a huge amount of data was been created by statistic, contact tracing, academic and medical research. How to merge these data, visualized, and make it understandable for the human being to analysis eventually. BI technology might be the way.

Existed Product

Many products were produced recently such as Microsoft BI, Oracle BI, IBM BI, ArcGIS(used to use by UK government)….. However, many of them require, more or less, some level of data science and programming background.

Take an example of Superset. An open-sourced BI web application built upon Python and JavaScript, created by Airbnb, incubating by Apache foundation. (https://airbnb.io/projects/superset/) This software was been comment as one of the easiest to use BI application. However, it is still, from the user experience aspect, it still does have a friendly learning-curve to use. One of my friend who working in a tech company comments that ‘Superset still is a tool build for developer and data sciences’. In his company, if a product manager wants to view a custom chart in BI, they ask a developer to make one for them.

Idea / Break Point

In my point of view, this can be improved. A BI tool should be easy to use with a friendly learning curve. So what I can do? I took an idea from Unreal Engine which is a game development tool I’ve used before. In the Unreal Engine, there is a visual scripting tool called: Blueprint. The Blueprint visual scripting tool use graph, node and connection line elements to let even a designer can quickly construct their concept scripting for the game. At least that is an image what they expecting.

So I think if that is possible bring the visual scripting concept into data science area. Here is some concept design draft I draw for this new system. The objective is: One with a very basic math background can create a complex data visualization graph.
