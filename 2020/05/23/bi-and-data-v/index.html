No-Code Concept

Overall

The ultimate gold for this idea is to try to find a way to let a user can build a data visualization format(chart, map, etc…) who without coding and data science background. Or maybe not completely solve this question but to bring it closer.

Background

The concept of No-Code is rising. Many software is targeting to deliver an easy-to-maintain application without coding skill required to their clients.
Like in this article: (https://medium.com/@rrhoover/the-rise-of-no-code-e733d7c0944d) Ryan said that “As creating things on the internet becomes more accessible, more people will become makers. It’s no longer limited to the <1% of engineers that can code resulting in an explosion of ideas from all kinds of people”. He listed a range of No-Code products and suggested that the No-Code concept is a new trend in software field.

Specifically, in the business area, a BI tool built upon the No-Code concept by Microsoft is getting popular.(https://www.zdnet.com/article/no-code-software-microsofts-power-apps-gets-better-teams-links-and-mixed-reality/). By using simple drag and drop editor, any user is able to build their own data visualization pages.

A Microsoft senior engineer says to a Chinese media that he believes No-Code or codeless will be a core element in the digital transformation. He defined a role who using these platform as “Power User”, like a senior Excel user or Powerpoint user, they are skilled working but not essentially a programmer or researcher. He mentioned four key points in the No-Code platform:

  1. The No-Code platform needs to solve most of the business need.
  2. The No-Code platform needs to easily merge and integrate into the target business’s current IT system.
  3. Consider the cost of using the No-Code platform carefully.
  4. Figure out who will be using this platform.

The Chinese media report(by CSDN): https://mp.weixin.qq.com/s/eO_B8u0M6HwUim9Yyrzs3g

In the industry, many companies starting to invest in No-Code platform like Voiceflow, a No-Code tool to build a Voice Assistant (https://www.voiceflow.com/). It seems that their client does not get satisfied by buying coded software and maintenance services, but they are more likely would purchase software they can control and use.

Business Intelligence

Data-driven business intelligence(BI) is a phenomenon topic along with data science and data visualization technology. When business dealing with big data, especially when the data is too big for any human can interpret without using a certain visualization method.

Despite there is a business in the word BI, this technology is not only work in the industrial. Let made an example of the COVID-19 epidemic, in the year of the Internet, a huge amount of data was been created by statistic, contact tracing, academic and medical research. How to merge these data, visualized, and make it understandable for the human being to analysis eventually. BI technology might be the way.

Existed Product

Many products were produced recently such as Microsoft BI, Oracle BI, IBM BI, ArcGIS(used to use by UK government)….. However, many of them require, more or less, some level of data science and programming background.

Take an example of Superset. An open-sourced BI web application built upon Python and JavaScript, created by Airbnb, incubating by Apache foundation. (https://airbnb.io/projects/superset/) This software was been comment as one of the easiest to use BI application. However, it is still, from the user experience aspect, it still does have a friendly learning-curve to use. One of my friend who working in a tech company comments that ‘Superset still is a tool build for developer and data sciences’. In his company, if a product manager wants to view a custom chart in BI, they ask a developer to make one for them.




Visual Scripting

In my point of view, this can be improved. A BI tool should be easy to use with a friendly learning curve. So what I can do? I took an idea from Unreal Engine which is a game development tool I’ve used before. In the Unreal Engine, there is a visual scripting tool called: Blueprint. The Blueprint visual scripting tool use graph, node and connection line elements to let even a designer can quickly construct their concept scripting for the game. At least that is an image what they expecting.
